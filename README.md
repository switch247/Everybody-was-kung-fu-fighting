# Everybody Was Kung Fu Fighting

## Description

"Everybody Was Kung Fu Fighting" is an exciting two-player fighting game inspired by classic martial arts films, featuring characters reminiscent of legends like Bruce Lee, Jackie Chan, and Kung Fu Panda. Players can engage in epic battles, showcasing unique fighting styles and special moves.

## Features

- **Two-Player Mode**: Challenge friends in local multiplayer or take on AI opponents.
- **Diverse Characters**: Select from a roster of fighters, each with unique abilities and signature moves.
- **Dynamic Arenas**: Battle in various beautifully designed arenas, each with distinct challenges.
- **Special Moves & Combos**: Master a variety of moves and combos to dominate opponents.
- **Engaging Soundtrack**: Enjoy an energetic soundtrack that captures the spirit of martial arts films.
